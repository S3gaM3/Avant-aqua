## Примечание по п. 10.8 договора (Internet Explorer)

Современные версии **WordPress** и **WooCommerce** не гарантируют работу в IE. Целевые браузеры для приёмки целесообразно согласовать как **две последние стабильные версии** Chrome, Edge, Firefox, Safari; мобильные — по тексту договора.
