# Чек‑лист финала, приёмки и передачи (Приложение 2 + п. 5 договора)

Использовать перед ссылкой на Акт сдачи‑приёмки и переносом на прод.

## Функционал по Приложению 1

- [ ] Главная: современный макет, оффер, преимущества, контакты (или ссылка на страницу контактов).
- [ ] Каталог: неограниченные категории/товары (smoke на пагинации и большом числе позиций по возможности).
- [ ] Сортировка по цене и новизне.
- [ ] Фильтры: цена, производитель, цвет, скидки.
- [ ] Карточка: галерея, описание, старая цена при скидке, наличие, «В корзину», «Купить в один клик», табы «Описание» и «Характеристики».
- [ ] Поиск по каталогу и по страницам/новостям.
- [ ] Три формы: обратная связь, запрос цены (с привязкой к товару), заказ как заявка — если отделён от checkout.
- [ ] Адаптив: проверка 320px, 768px, 1024px, 1920px (Прил. 2).
- [ ] ЧПУ, мета‑заголовки/описания шаблонов, `sitemap.xml`, `robots.txt`.
- [ ] Админка: роли для сотрудников Заказчика без лишних прав; **инструкция** отправлена на e‑mail.
- [ ] Корзина и оформление: имя, телефон, e‑mail, комментарий; письма о новом заказе доходят (SMTP).
- [ ] ЮKassa: только если есть письменная заявка и ключи.

## П. 1.1 договора (типы контента)

- [ ] Страницы с текстом и HTML, новости/статьи, галерея, «Напишите нам», карта сайта XML, поиск.
- [ ] По запросу: регистрация в Яндекс/Google/Mail.ru (вебмастеры), Метрика, Jivo.

## Перенос на хостинг и домен Заказчика

- [ ] Экспорт/импорт БД или миграция плагином; одинаковые версии PHP не ниже требований WooCommerce.
- [ ] `wp-config.php`: уникальные ключи salts, `WP_DEBUG` выключен на проде.
- [ ] HTTPS, редирект с HTTP, актуальные права на файлы.
- [ ] Cron (wp‑cron или системный) для задач WooCommerce.
- [ ] Снятие `noindex` с продакшена после согласования.

## Пакет передачи исходников (по запросу п. 2.3–2.4, 10.5)

- [ ] Архив `wp-content` (тема `avant-shop-child`, плагин `avant-site-core`, список остальных плагинов с версиями).
- [ ] Дамп базы данных.
- [ ] Краткий файл `STACK.md` или перечень плагинов с версиями внутри архива.
- [ ] Доступы к хостингу и админке — по отдельному защищённому каналу после Акта.

## Юридическое и эксплуатационное

- [ ] На сайте размещены документы по требованиям Заказчика (оферта, политика персональных данных — по их шаблонам).
- [ ] Согласован перечень браузеров для приёмки (учесть п. 10.8 договора и реальную поддержку WP/Woo).

## Тест после переноса

- [ ] Оформление тестового заказа с продакшена.
- [ ] «Один клик» на реальном товаре.
- [ ] Отправка каждой формы на целевой ящик.
