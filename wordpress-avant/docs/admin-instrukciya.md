# Краткая инструкция для администратора сайта (WordPress + WooCommerce)

Документ для передачи Заказчику по договору (настройка админ‑панели, Приложение 2).

## Вход и роли

- Адрес входа: `https://ваш-домен.ru/wp-admin/`.
- Для сотрудников, которые только наполняют каталог, используйте роль **Редактор** или **Менеджер магазина** (Shop manager), не **Администратор**, если это не требуется.

## Товары и каталог

- **Товары → Все товары** — создание и редактирование карточек.
- **Товары → Категории** — дерево разделов витрины.
- **Товары → Атрибуты** — задайте атрибуты **Производитель** и **Цвет** (или иные по ТЗ), затем привяжите их к товарам на вкладке «Атрибуты» в карточке.
- Цена со скидкой: поля обычная цена и **цена распродажи**; наличие — поле «Запасы» при включённом учёте остатков.

## Главная страница и слайдер

- **Внешний вид → Настроить → раздел «Главная страница»** (если добавлен в теме): до трёх изображений слайдера.
- Тексты оффера и преимуществ можно вынести в **Страницы** или оставить в шаблоне темы — по согласованию с разработчиком.

## Оформление заказа

- Страница «Оформление заказа» должна использовать **классический шорткат** `[woocommerce_checkout]`, если применён упрощённый набор полей из темы.
- Проверьте доставку: при физических товарах может понадобиться полный адрес — тогда отключите упрощение (см. `implementation-next-step.md`).

## Формы обратной связи и заявок

- Настраиваются в плагине форм (см. [plugin-stack.md](plugin-stack.md)): отдельные формы для обратной связи, запроса цены (скрытое поле с ID товара), при необходимости — заявки.

## Поиск и фильтры

- После установки плагинов поиска и фильтров следуйте их мастерам индексации; ЧПУ — **Настройки → Постоянные ссылки** (рекомендуется «Название записи»).

## SEO

- Мета‑заголовки и описания для типов страниц — в SEO‑плагине (Rank Math / Yoast). Карта сайта и `robots` — там же или в ядре WP.

## Тестовый стенд

- В настройщике темы включите **«Закрыть сайт от индексации»** на демо‑домене; на продакшене перед запуском — **выключите**.

## «Купить в один клик» и оплата

- Плагин **Avant Site Core** создаёт заказ в WooCommerce; в настройках WooCommerce должны быть включены простые способы оплаты (например, оплата при получении или безнал), пока не подключена ЮKassa.

## Почта

- Настройте SMTP (см. [plugin-stack.md](plugin-stack.md)), иначе письма о заказах могут не доходить с хостинга Заказчика.

## Локальный просмотр без Docker

- Используйте **Local**, **Laragon**, **OpenServer**, **XAMPP** или любой другой стек с PHP и MySQL: установите WordPress, затем скопируйте тему и плагин из `wordpress-avant/wp-content/` (см. раздел **«Локально посмотреть (без Docker)»** в [README.md](../README.md)).
