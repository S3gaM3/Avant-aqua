# Согласованный стек плагинов (рекомендация к фиксации с Заказчиком)

Один набор на роль, без дублирования функций. Конкретные slug на хостинге Заказчика проверить по совместимости с версией PHP и кэшем.

## Обязательные для ТЗ (Приложение 1)

| Роль | Рекомендуемый вариант | Альтернатива | Заметки |
|------|----------------------|--------------|---------|
| Магазин | **WooCommerce** | — | Ядро каталога, корзины, заказов, писем. |
| Фильтры каталога (цена, производитель, цвет, скидки) | **Filter Everything PRO** или бесплатный **Filter Everything** при достаточном функционале | **YITH WooCommerce Ajax Product Filter** | Нужны чекбоксы/диапазон цены и условие «распродажа». |
| Поиск по сайту и каталогу | **FiboSearch** (AJAX Search for WooCommerce) | **Ivory Search** + донастройка индекса товаров | Единая строка поиска, автодополнение. |
| Формы (ОС, запрос цены, заказ‑заявка) | **Fluent Forms** | **WPForms Lite**, **Contact Form 7** | Три формы, разные получатели; для «запрос цены» — скрытое поле `product_id`. |
| SEO (ЧПУ, мета, sitemap, robots) | **Rank Math SEO** | **Yoast SEO** | Один SEO‑плагин; XML‑карта и robots — через него или ядро WP. |
| Доставляемость почты | **WP Mail SMTP** | Транспорт SMTP хостинга без плагина | Критично для «заказы на почту». |

## По запросу Заказчика (договор п. 1.1, 2.7, 2.9)

| Роль | Рекомендация |
|------|----------------|
| Яндекс.Метрика | Вставка счётчика в `<head>` или плагин вроде **Yandex.Metrica** / произвольный «Insert headers». |
| JivoSite | Официальный плагин **JivoChat** или вставка кода. |
| ЮKassa | Официальный **ЮKassa для WooCommerce** после договора с эквайером и выдачи ключей. |

## Эксплуатация (снижение рисков срыва сроков)

| Роль | Рекомендация |
|------|----------------|
| Резервные копии | **UpdraftPlus** или бэкап панели хостинга. |
| Кэш | Серверный кэш хостинга; если плагин — **LiteSpeed Cache** (при LiteSpeed) или **WP Rocket**, с исключением корзины/checkout и cookie сессии. |
| Закрытие теста от индексации | **Yoast** / **Rank Math** режим «не индексировать» или плагин типа **Discourage Search Engines** на стенде. |

## Не устанавливать в рамках договора

Интеграции с CRM/1С, SEO‑продвижение как услуга, тяжёлые конструкторы страниц без отдельного решения Заказчика.
